## 🎯 Guia de Uso dos Templates

### 📋 **Como Usar Este Sistema**
1. **Copie** o template apropriado para uma nova nota
2. **Renomeie** com a data (ex: "2024-08-05 - Treino Segunda")
3. **Preencha** durante ou após o treino
4. **Linke** de volta para [[📈 Dashboard de Progresso]]

### 🔄 **Frequência de Uso**
- **Template Treino Força:** 3x/semana (Seg, Qua, Sex)
- **Template Treino Cardio:** 2x/semana (Ter, Qui)  
- **Template Dia Ativo:** 1x/semana (Sáb)
- **Template Recuperação:** 1x/semana (Dom)
- **Template Avaliação Semanal:** 1x/semana
- **Template Avaliação Mensal:** 1x/mês
